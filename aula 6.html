Fórmulas de Recorrência

Conceitua-se fórmula recorrente uma fórmula onde um de seus termos é a própria fórmula gerando assim uma Recursão ou Recorrência.

Sendo assim, existem métodos utilizados para eliminar a recorrência das fórmulas, os quais serão estudados abaixo:

  • Método da Soma

  • Método da Substituição

  • Método da Árvore

  • Método Mestre

Método da Soma

O método da soma consiste em, literalmente, somar as N funções descendentes da função inicial, ou seja, gerar resultado a partir das variações de F(N). Por exemplo:

Para F(N) = F(N-1) + 1, onde F(1) = 1 temos:

T(N) = T(N-1)+1

T(N-1) = T(N-2)+1

T(N-2) = T(N-3)+1

...

T(2) = T(1)+1

T(1) = 1

+_______________

T(N) = 11 + 12 + 13..... 1N

T(N) = N

T(N) Є O(N)

Método da Substituição

O método da substituição é também conhecido como expandir, conjecturar ou verificar e consistem em duas etapas:

  • Pressupor a fórmula solução
  • Através da indução matemática provar que a solução funciona

O nome é originado do fato de que o palpite será substituído pela resposta final, sendo assim o palpite poderá ser alterado para chegar a respostas mais precisas ao verificar a corretude desse palpite

Provar {T(N) | N >= 1}

Utilizando a base T(1)

e Supondo T(N)

Provamos por indução T(N-1)

Segue exemplo com o cálculo recursivo exponencial(a^b):

Tendo T(N)1 = 4 + T(N -1)

Chegamos a fórmula T(N)2 = 4N - 2

Para N = 1 é fácil ver que a solução palpite está correta, pois T(1)2 = 2.

Agora, se tivermos um N > 1 vamos supor que T(N)2 também é válida para (N-1) ao invés de (N)

T(N)1 = 4 + T(N-1)2

T(N)1 = 4 + [4(N - 1) - 2]

T(N)1 = 4 + [4N - 4 - 2]

T(N)1 = 4 + [4N - 6]

T(N)1 = 4N - 2

T(N)1 = T(N)2

T(N)1,2 Є O(N)

Método da Árvore

Árvore de recursão, também chamado, é utilizado para se fazer uma suposição mais um pouco mais precisa do que um chute ao se utilziar o método da substituição, ou seja, é pouco preciso se comparado ao método da soma ou o próprio método da substituição.

Consiste em simular os nós de uma chamada recursiva desmembrando cada uma delas até N atingir o F(1) = 1.

Observe o exemplo para F(N) = 2F(N/2) + N, para F(1) = 1:

1.N

/\

2.N/2 N/2

/\/\

3.T(N/4)T(N/4)T(N/4)T(N/4)

...

Se somarmos cada linha teremos:

1.: N = N

2.: N/2 + N/2 = N

3.:N/4 + N/4 + N/4 + N/4 = N

.... = N

+_______________

T(N) = N1 + N2 + N3 .... + NLOG(N)

T(N) = N * Log(N)

F(N) Є O(N Log(N))

Método Mestre

O método mestre costuma ser usado para responder problemas de recorrência onde a fórmula de recorrência apresenta um formato específico:

T(N) = AT(N/B) + F(N)

  • Onde A >= 1
  • B > 1
  • A, B são constantes
  • e F(N) é uma função positiva

Obedecendo as regras acima temos que comparar F(N) com os três casos. ɛ > 0

Obs.:levar em consideração que ɛ é uma constante muito pequena e ɛ > 0

  • Caso 1:

    Se F(N) = O(N^LogB(A - ɛ)), então T(N) Є ϴ(N^LogB(A)).

  • Caso 2:

    Se F(N) = ϴ(N^LogB(A)), então T(N) Є ϴ(N^LogB(A) * Log2(N)).

  • Caso 3:

    Se F(N) = Ω(N^LogB(A + ɛ)), e se A * F(N/B) < c * F(N) para c < 1, então T(N) então T(N) Є ϴ(N^LogB(A)) Є ϴ(F(N)).

1. T(N) = 9T(N/3) + N

A = 9; B = 3; F(N) = N

N^Logb(A) = N^Log3(9) = N^(2 - ɛ)

Caso 1!

T(N) Є ϴ(N^LogB(A))

T(N) Є ϴ(N^2)

2. T(N) = T(2N/3) + 1

Caso 2!
